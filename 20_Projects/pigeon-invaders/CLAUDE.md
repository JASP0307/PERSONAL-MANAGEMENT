# Pigeon Invaders

**Status:** active
**Area:** 20_Projects

## Purpose
DIY automated laser bird deterrent for a residential complex.

## Current state
Prototype 1 (250mW red laser) tested: partially effective, but the red dot washes
out on the white building in sunlight. Moving to v2: 300mW green laser (on order),
new 3D CAD design for the mount, stronger MG996R servos for the pan-tilt gimbal,
and a custom PCB for the main circuit. Not yet installed on-site. See NOTES.md
for the log.
